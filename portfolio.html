INAI ProDatos

Tipo de Desarrollo: Consultora (Indra)

Sitio Web para INAI (Instituto Nacional de Acceso a la Información).

Este desarrollo consistio en una plataforma para denunciar el mal uso de los datos personales, fue una integración de tecnologías: SAP, OpenText y .NET en la cual participe en el Front-End de la plataforma en ASP y manejando los webservices de SAP y OpenDocument que están basados en SOAP y WCF respectivamente, para integrar subida de archivos, firmado digital y la interacción de los casos en SAP CRM.

Skills utilizadas:
Manejo de ASP, Base de Datos en SQL Server 2008 con ORM nHibernate, utilización de Web Services para comunicarse con Open Text y SAP (SOAP y WCF), CSS, implementación de medidas de seguridad OWASP.
